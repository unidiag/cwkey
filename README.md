![CW Key](https://github.com/unidiag/cwkey/blob/main/pic.jpg)

  Возможности:
  - При включении: `HI` (.... ..) на последней сохранённой скорости (только на динамик, без выхода)
  - Полностью повторяет все возможности автоматического телеграфного ключа, выпускаемого в 80-е годы в СССР https://www.radionic.ru/node/1026
  - Регулировка скорости WPM от 5 до 35 слов в минуту (достигается удержанием соответствующей кнопки "+" или "-"). При включении скорость восстанавливается из последней установленной до выключения.
  - Регилировка тона в пьезе самоконтроля от 100 до 2500 гц (достигается нажатием "тире" и соответствующей кнопкой "+" или "-")
  - Возможность отключить звук самоконтроля (зажать "тире" + "-" до момента пока тон не станет 0 Гц). При включении тон всегда включён.
  - Зажать левый ручаг и "-" -- положение рычагов для левой руки, если "+" -- для правой. Ориентация по классике - большой палец для "тире".
  - При коротком нажатии на "+": `CQ CQ CQ DE EW6ML EW6ML EW6ML K`
  - При коротком нажатии на "-": `TNX QSO 73`

Компилировать в Keil 5.
Прошивать контроллер через STC-ISP.exe
